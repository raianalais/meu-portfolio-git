O conflito ocorreu no arquivo main.cpp, na função de saudação, onde duas branches diferentes alteraram a mesma linha.

A branch feat/conflito-a definiu a mensagem como "Versao A da saudacao", enquanto a branch feat/conflito-b definiu como "Versao B da saudacao".

Para resolver o conflito, decidi criar uma nova versão combinando as duas ideias.

O conflito foi resolvido manualmente editando o arquivo, removendo os marcadores de conflito e realizando um novo commit com git add e git commit.
